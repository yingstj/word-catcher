# word-catcher
  Words fall from the sky. Catch the right ones. A reading game for 5-year-olds learning to decode sentences.

Word Catcher is a reading comprehension game for early readers. Kids decode words in a sentence, then catch the matching pictures as they rain down from the sky.

How it works: A sentence appears at the top of the screen — something like "The apple falls from the tree onto the grass." The target words (apple, tree, grass) are highlighted. Then pictures start falling: 🍎 🌳 🌱 mixed in with distractors like 🍃 🐦 ☁️ ☀️ — items that fit the world but aren't in the sentence. The kid has to read each word, hold its meaning, and tap the matching picture when it falls. No labels on the pictures. No audio. Pure word-to-meaning-to-image decoding. When all the target pictures are caught, the next sentence loads.

Themed worlds: Each round takes place somewhere familiar such as a 🏫 School · 🏖️ Beach · 🌳 Backyard · 🍳 Kitchen · 🛏️ Bedroom · 🧸 Playroom · 🛝 Playground · 🦆 The Park. Finish all the sentences in a theme to unlock an animal friend. Collect them all.

Designed for 5-year-olds

No losing. Wrong taps wobble and shake but never penalize.
No timer. Items fall slowly and sway as they go.
No audio. This is a reading comprehension exercise, not listening practice.
Visual feedback only. When you catch a target, the word in the sentence flashes yellow and turns green — confirming the match.
Forgiving spawn rate. Items appear at a pace that doesn't overwhelm.

Tech
One HTML file. No build step, no dependencies beyond Google Fonts (Fraunces + Nunito). Just open index.html and play, or host it anywhere that serves static files.
Works on phones, tablets, and laptops.

Clone the repo and open index.html in any browser. That's the whole setup.
